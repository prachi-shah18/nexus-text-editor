# Nexus Text Editor

Nexus Text Editor is a professional desktop-based text editing application developed using Java Swing and Object-Oriented Programming principles.

The application provides an elegant user interface, intelligent document management, modern editing tools, and responsive desktop performance.

---

## Features

- Modern Java Swing User Interface
- Dark Mode Support
- Intelligent Undo / Redo System
- Auto Save Functionality
- File Open & Save Operations
- Search Functionality
- Real-Time Status Bar
- Keyboard Shortcuts
- Responsive Desktop Experience
- Clean OOP Architecture

---

## Technologies Used

- Java
- Swing Framework
- Object-Oriented Programming
- Event Handling
- File Handling
- Java AWT

---

## Application Modules

### File Management
Handles opening, saving, and creating text documents securely.

### Text Editing Engine
Provides editing features with intelligent state management.

### Theme Controller
Manages modern light and dark interface modes.

### Event Management System
Processes keyboard, mouse, and UI interactions efficiently.
